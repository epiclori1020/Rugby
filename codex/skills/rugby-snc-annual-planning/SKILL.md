# Rugby S&C Annual Planning

## Goal

Build a rugby-specific one-year S&C plan for Rugby Donau's U22/development pathway.

## Required Context

Read first:

- `AGENTS.md`
- `docs/00_project_context.md`
- `plans/00_planning_principles_from_research.md`
- `plans/01_injury_prevention_framework.md`
- `plans/02_periodization_framework.md`
- `plans/03_position_clusters_and_modules.md`
- `plans/04_returner_integration_framework.md`

## Planning Rules

- Two team S&C sessions per week by default.
- Whole-body sessions, not bodypart splits.
- Tuesday is the main stimulus unless the game calendar says otherwise.
- Thursday is a low-volume, high-quality primer when Saturday games are likely.
- Do not copy professional volume.
- Use position clusters:
  - Collision Forwards.
  - Hybrid.
  - Speed/Space Backs.
- Include testing and deload windows.
- Include August bridge:
  - week 1 reset/deload.
  - week 2 maintenance.
  - week 3 re-entry preparation.

## Output Pattern

1. Assumptions.
2. Macrocycle table.
3. Phase priorities.
4. Testing windows.
5. Deload/reset windows.
6. Tuesday/Thursday logic per phase.
7. Open data still needed from the club.

## Safety Checks

- Speed exposure retained year-round.
- Strength signal retained year-round.
- Contact load is separately planned.
- Injury-prevention microdoses remain year-round.
- Returners are not folded into the normal plan without caps.
