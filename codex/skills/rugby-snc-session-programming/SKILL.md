# Rugby S&C Session Programming

## Goal

Write practical sessions that can be run on field and in a small gym.

## Default Session Order

1. Pre-session monitoring.
2. Activate/RAMP prep.
3. Speed or power while fresh.
4. Main strength.
5. Position cluster module.
6. Conditioning or rugby-specific field work if appropriate.
7. Robustness/cooldown.
8. Post-session sRPE and contact index.

## Position Clusters

Collision Forwards:

- Front Row.
- Locks.
- Neck/brace/sled/set-piece bodyline.

Hybrid:

- Back Row.
- Centres.
- Accel-to-contact, decel/re-accel, adductor/shoulder.

Speed/Space Backs:

- Halves.
- Back Three.
- Max velocity, fly-ins, calf/hamstring/foot, decel footwork.

## Constraints

- Small gym: plan pod rotations.
- Players work or study: avoid excessive junk volume.
- Thursday before Saturday game: finish sharp, not tired.
- Do not add full contact casually.
- No repeated-sprint/RHIE punishment blocks unless phase and readiness justify it.

## Output Pattern

- Session goal.
- Duration.
- Equipment.
- Team-wide blocks.
- Position-cluster modifications.
- Green/yellow/red adjustments.
- Returner alternatives.
- Monitoring notes.
