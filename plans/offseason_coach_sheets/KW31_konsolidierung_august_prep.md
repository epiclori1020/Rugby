# Coach-Sheet KW31: Konsolidierung + August Prep

Stand: 2. Juni 2026

Zeitraum: 28. und 30. Juli 2026

Ziel: sauberer Abschluss vor Arwins Abreise, keine Belastungsspitze, August-Uebergang vorbereiten.

Hauptregler der Woche: Konsolidierung, keine deutliche Steigerung.

## Di 28. Juli: Session 6A

| Block | Inhalt |
| --- | --- |
| Check-in | Ampel + Urlaubs-/August-Verfuegbarkeit |
| Prep | RAMP, Sprintmechanik, Landing, Neck/Shoulder |
| Mini-Recheck optional | Broad Jump 2 Versuche + 20-30 m Sprint 1-2 Versuche, nur wenn Gruppe frisch |
| Power | Med-Ball Throw 3x3 |
| Pod A | Hauptlift 3x4 @ RPE 7, kein Grind |
| Pod B | Push/Pull 3x6 |
| Pod C | Unilateral + Carry 2-3 Runden |
| Microdose | Nordic 2x3-4, Copenhagen/Adductor 2x15 s, Calf 2x12 |
| Conditioning | 2 Serien Ball-in-Play oder 6x60 m Tempo, je nach Readiness |
| Abschluss | August-Regeln kurz erklaeren |

Mini-Recheck:

- nur wenn Gruppe frisch, regelmaessig trainiert und organisiert ist.
- kein harter Re-Test.
- keine neuen Tests.

## Do 30. Juli: Session 6B

Ziel: bekannte Uebungen festigen, Spieler auf August vorbereiten.

| Block | Inhalt |
| --- | --- |
| Check-in | Ampel, offene Returner-/Schmerzinfos |
| Prep | kurze RAMP-Variante, Mobility, Neck/Shoulder |
| Speed | 3x20 m smooth fast, keine Max-Jagd |
| Power | leichte Med-Ball Throws 2-3x3, CMJ + Stick 2x2 |
| Krafttechnik | bekannte Hauptmuster 2-3x5 @ RPE 6 |
| August Prep | Technik-Review: Squat/Hinge/Push/Pull/Carry, simple Laufoptionen |
| Robustheit | Hamstring/Adductor/Calf/Shoulder/Trunk kurze Runde |
| Conditioning | optional 6-8 min locker, kein harter Abschluss |
| Abschluss | Session-RPE, August-Verfuegbarkeit, klare No-Gos |

## No-Gos vor August

- keine neue komplexe Uebung.
- keine harte Kontaktspitze.
- keine Max-Sprints unter Ermuedung.
- keine letzte "Test dich aus"-Einheit.
- keine verpassten Einheiten nachholen.
