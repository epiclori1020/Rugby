## Legende

- CF = Collision Forwards, HY = Hybrid, SB = Speed/Space Backs.
- RPE = Belastungsgefuehl fuer Satz/Uebung von 1-10.
- sRPE = Gesamtbelastung nach der Einheit von 0-10.
- Quality = sauber, schnell und technisch gut; nicht maximal ermuedend.
- TUT = Time under tension, also kontrollierte Zeit unter Spannung.
- Med-Ball = Medizinball.
- Bronco = Shuttle-Conditioning-Test ueber 20/40/60 m, nur optional.
- Returner = Spieler nach Verletzung oder mit unklarem Belastungsstatus.

## Immer gleich

- Check-in: Da, Ampel, Schmerz/Returner.
- Speed/Power frisch, volle Pause, keine Ermuedungsjagd.
- Hauptlift dokumentieren: Last, Reps, RPE.
- Gelb: 30-50 Prozent weniger Volumen, RPE 5-6, keine Zusatzkondition.
- Rot: kein normales Training.
- Wenn chaotisch: Conditioning kuerzen, Speed/Power-Qualitaet behalten.
- Abschluss: sRPE, Pain/Issue, Follow-up.
- Varianten: A+ fuer starke gruene Spieler, A normal, B organisatorisch/equipment, C gelb, D Returner/unklar.

# KW25 Einheit 1: Baseline

Datum: Di 16. Juni oder Do 18. Juni 2026

Ziel: Spieler kennenlernen, Ampelstatus, Power, Sprint, optional Conditioning.

Material: Klemmbrett, Datensheet, Stifte, Waage, Massband, Huetchen, Stoppuhr, Smartphone/App, Stativ wenn verfuegbar, Med-Ball, optional Bronco-Linien.

## Ablauf

| Zeit | Block | Inhalt |
| --- | --- | --- |
| 0-10 | Check-in | Name, Position, Masse, Readiness 1-5, Schmerz/Ort, Returner |
| 10-22 | Warm-up | Laufvarianten, A-March/A-Skip, Dribble, Pogo, Snap-down, 2-3 progressive Accels |
| 22-37 | Screening | Adductor, Ankle/Calf, Shoulder, Bridge/Pelvis, Neck Symptomcheck |
| 37-50 | Power Rotation | Gruppe A Broad Jump 2-3 Versuche, Gruppe B Seated Med-Ball Throw 2-3 Versuche; dann Wechsel |
| 50-68 | Sprint | 30 m, 2-3 Versuche, Splits nur bei stabilem Setup |
| 68-85 | Bronco optional | nur wenn Gruppe, Wetter, Platz und Messqualitaet passen |
| 85-90 | Abschluss | sRPE, Pain/Issue, Follow-up |

## Notizen

| Spieler / Gruppe | Auffaelligkeit | Entscheidung |
| --- | --- | --- |
| | | |
| | | |
| | | |

Stop: kein Bronco bei Chaos, Hitze, vielen gelben Spielern oder schlechter Messqualitaet.
