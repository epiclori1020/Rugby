# Rugby Donau Exercise Pool

Zweck: Dieses Dokument ist der praktische Uebungspool fuer die spaetere Jahresplanung. Es ist kein starres Menue. Pro Einheit werden wenige Bausteine aus den Familien gewaehlt.

## Cluster

| Cluster | Positionen | Schwerpunkt |
| --- | --- | --- |
| CF - Collision Forwards | Front Row, Locks | Kraft, horizontale Kraft, Neck/Shoulder/Trunk, Sled, Contact Prep |
| HY - Hybrid | Back Row, Centres | Kontakt plus Speed/COD, Decel, Adduktoren, Shoulder |
| SB - Speed/Space Backs | Halves, Back Three | Acceleration, Max-V, RSA, Decel, Calf/Hamstring/Foot |

## Warm-up / Prep

| Uebungsfamilie | Beispiele | Einsatz |
| --- | --- | --- |
| RAMP locomotion | shuffle, skip, backpedal, curves | jede Einheit |
| Sprint mechanics | hip-lock, A-march, dribble-run | vor Speed |
| Ankle stiffness | ankling, pogo, line hops | Speed/COD-Tage |
| Landing | snap-down + stick, squat-to-stick | COD/Power |
| Plant/Cut prep | plant-and-hold 45/90 Grad | COD/Agility |
| Neck/Shoulder | neck iso, band ER, push-up plus | jede Einheit klein dosiert |

## Speed / Acceleration

| Uebungsfamilie | Beispiele | Cluster-Farbe |
| --- | --- | --- |
| Starts | falling start, rocker start, 2-point, split-stance | alle |

| Sled acceleration | heavy sled 10-20 m, resisted march | CF > HY > SB |
| Max-V exposure | build-up + fly, 30-50 m smooth fast | SB > HY > CF |
| Mechanics | wickets, dribble wickets, arc sprint | SB/HY |

## Deceleration / COD / Agility

| Uebungsfamilie | Beispiele | Progression |
| --- | --- | --- |
| Linear decel | decel zone, accel-decel-hold | slow -> faster |
| Planned cut | 45 Grad, 90 Grad, 180 Grad | planned -> faster |
| Lateral plant | shuffle to plant-and-hold | static -> run-in |
| Reactive | colour cue, mirror, 2-option evade | 2 options -> rugby-nah |

## No-Go als Standard

- Overspeed fuer Development-Spieler ohne stabile Speed-Historie.
- Max-V unter Ermuedung.
- hohe Depth-Jump-Volumina.
- volle Olympic Lifts vom Boden fuer den ganzen Kader.
- neck bridges als Pflicht.
- grosse Live-Wrestling-/Full-contact-Bloecke im S&C.
- harte Conditioning-Strafen nach schlechter Technik.
- regelmaessiges maximales 1RM-Testen.
