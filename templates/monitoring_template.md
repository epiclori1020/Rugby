## Post-Session Check

Maximal 3 Angaben:

| Angabe | Format | Wert |
| --- | --- | --- |
| Session-RPE | 0-10 |  |
| Dauer | Minuten |  |
| Pain/Issue nach Training | 0-10 + Ort/Notiz |  |

Kontaktindex:

- 0 = kein Kontakt.
- 1 = Kontaktvorbereitung / Bags / Technik ohne echte Live-Kollision.
- 2 = kontrollierter Live-Kontakt, klar begrenzte Drills.
- 3 = echter Vollkontakt / mehrere Live-Bloecke / intensives Set Piece.

Hinweis:

- Der Kontaktindex wird coachseitig im Wochenboard gefuehrt, nicht als Spieler-Pflichtfeld.
- Speed-Exposures werden ebenfalls coachseitig geplant und geloggt.

## Ampel

Gruen:

- Schmerz 0-2/10.
- keine relevanten Flags.
- geplantes Training.

Gelb:

- Schmerz 3-4/10 oder ein moderater Flag.
- Volumen 30-50 % reduzieren.
- keine Zusatzkonditionierung.
- Kontakt/Sprint/COD-Dichte reduzieren.

Rot:

- neuer Schmerz >4/10.
- Kopf-/Nackensymptome.
- veraenderte Lauf-/Kontaktmechanik.
- zwei oder mehr gelbe Flags.
- kein Vollkontakt, kein Max-Speed, kein schweres Unterkoerperheben.
- medizinische/therapeutische Ruecksprache.
